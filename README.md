# CloudeTest1

A learning repository for getting started with **Claude Code** — Anthropic's AI-powered CLI for developers.

## Contents

### `claude-code-study-plan.html`

A self-contained, dark-themed HTML study guide for developers new to Claude Code.

**Structure:** 4 weeks · 20 days · 1–2 hours per day

| Week | Theme | Highlights |
|------|-------|------------|
| 1 | Foundations | Installation, first session, slash commands, reading code |
| 2 | Core Skills | File editing, git integration, debugging, CLAUDE.md |
| 3 | Power Features | VS Code extension, custom commands, MCP servers, hooks |
| 4 | Capstone | Full project, testing, docs, reflection & habits |

- Milestone badges at the end of each week
- Tips for Success section with 6 best-practice cards
- Essential commands cheatsheet
- Zero external dependencies — single file, works offline

## Usage

Open the study plan directly in your browser:

```bash
open claude-code-study-plan.html
```

Or serve it locally:

```bash
python3 -m http.server 8080
# then visit http://localhost:8080/claude-code-study-plan.html
```

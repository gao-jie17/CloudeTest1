Foundations & First Steps

Get Claude Code installed, understand what it is, and run your first real interactions.

Beginner

  • Read the official Claude Code overview
  • Understand the difference vs chat UI
  • Learn the CLI-first philosophy

Installation & Setup
  • Install via npm i -g @anthropic-ai/claude-code

Your First Session

Tips for Success

🎯

Stay Specific

Vague prompts produce vague results. Tell Claude exactly which file, function, or behavior you want to change.

🔍

Review Every Change

Always read the diffs before accepting them. AI code can be subtly wrong. You are the final reviewer.

📄

Write a Good CLAUDE.md

Invest time upfront describing your stack, conventions, and preferences. It pays dividends throughout the project.

⚡

Small, Iterative Tasks

Break large requests into smaller steps. Each interaction should have a single, clear goal.

🧪

Test as You Go

Run your tests after each Claude-assisted change. Catch regressions early rather than debugging later.

📚

Read the Docs

Claude Code evolves quickly. Check the official docs and release notes regularly for new features.

Essential Commands Cheatsheet

claude Start interactive session
claude "your task" One-shot task from shell
/help Show all slash commands
/clear Clear conversation context
/compact Compress context to save tokens
/cost Show token usage & cost
/plan Enter planning mode
/memory View & manage saved memories
--print Non-interactive (CI/scripts) mode
--dangerously-skip-permissions Auto-accept all tool calls
CLAUDE.md Project-level persistent instructions
.claude/commands/ Custom slash command folder
